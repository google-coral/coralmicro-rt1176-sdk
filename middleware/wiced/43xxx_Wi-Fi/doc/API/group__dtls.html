wiced_dtls_security_type_t  type,
void *  data

Initialises a DTLS identity using a supplied certificate and private key.

Parameters
[out]identity: A pointer to a wiced_dtls_identity_t object that will be initialised
[in]type: type of security ( PSK or NONPSK ) being used.
[in]data: pass client identity-key pair for PSK and certificate-key information for NONPSK. In case of PSK, this API can be used for adding one entry. To add more than one entry into list, use wiced_dtls_add_psk_identity API.
Returns
wiced_result_t
