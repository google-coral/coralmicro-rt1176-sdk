Detailed Description

This library implements core WICED Audio APIs for audio capture and playback and managing resources used by the I2S audio hardware found on Wiced Audio Edition (WAE) Platforms.

Create the buffer used by the audio driver.

Note
If allocator is null and buffer_ptr_aligned is null, then the buffer will be allocated and owned by the audio driver. Not all platforms support using an allocator function.

[in]size: Size of the audio buffer - must be a multiple of the period size.

[in]allocator: Optional pointer to allocation function to use for allocating audio buffer.
