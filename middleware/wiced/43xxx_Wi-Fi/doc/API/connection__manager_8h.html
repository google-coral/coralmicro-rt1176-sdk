Detailed Description

Defines functions to access WiFi connection manager.

Following features are supported: Wi-Fi Direct Group Owner: Persistent group Wi-Fi Direct Client mode Wi-Fi WPS Registrar mode Wi-Fi WPS Enrollee mode
